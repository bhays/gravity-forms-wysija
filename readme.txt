== Description ==

Create Wysija Newsletter subscribers from Gravity Form entires.

Requires Gravity Forms version 1.6.10 or higher.

Thanks to Dave at Deckerweb for i18n enhancements and German translation

== Installation ==

1. Install as a regular WordPress plugin
2. Create a form with the appropriate fields via Gravity Forms
3. Setup Wysija Newsletters with one or many lists
4. Navigate to Forms->Wysija to setup feeds linking your forms and lists
